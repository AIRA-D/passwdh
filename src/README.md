# PasswdH

Консольная утилита на Rust для генерации и аудита паролей.

## Возможности
- **Безопасность**: Использует `zeroize` для очистки памяти.
- **Гибкость**: Поддержка масок (латиница, кириллица, цифры, спецсимволы).
- **Аналитика**: Расчет реальной энтропии с учетом паттернов (повторы, последовательности).

## Установка
```bash
cargo build --release
```

## Использование

- Генерация пароля длиной 16 с цифрами и спецсимволами:
```bash
./passwdh generate -e 16 -p lns
```
- Проверка существующего пароля:
```bash
./passwdh check "MyP@ss123" -p lbns -e 8
```
